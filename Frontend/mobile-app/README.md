# Mobile App

## Propósito

Aplicación móvil Flutter para usuarios de TravelHub.

## Stack

- Flutter
- Dart

## Requisitos

- Flutter SDK
- Dart SDK
- Dependencias de `pubspec.yaml`

## Configuración

La aplicación carga variables desde `.env`.

## Ejecución local

```bash
flutter pub get
flutter run
```

## Pruebas

```bash
flutter test
```

## Estructura relevante

- `lib/`: código de aplicación
- `test/`: pruebas
- `docs/`: guías específicas del proyecto

## Dependencias con backend

Consume APIs de autenticación, reservas y servicios de consulta expuestos por backend.
