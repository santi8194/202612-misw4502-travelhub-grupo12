# Partner Web

## Propósito

Aplicación web para gestión operativa de partners hoteleros.

## Stack

- Angular
- Cypress

## Requisitos

- Node.js
- Dependencias de `package.json`

## Configuración

La configuración se mantiene dentro del proyecto frontend.

## Ejecución local

```bash
npm install
npm start
```

## Pruebas

```bash
npm test
npm run e2e
```

## Estructura relevante

- `src/`: aplicación Angular
- `cypress/`: pruebas E2E

## Dependencias con backend

Consume principalmente `authservice` y `partner-management`.
